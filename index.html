Ultilizando Arrays

O que é uma array? Arrays são geralmente descritas como "lista de objetos"; elas são basicamente objetos que contem múltiplos valores armazenados em uma lista.

Para visualizar as soluções dos comandos, é só ir no console do pc.

Vou colocar aqui os scripts que estou praticando.

Arrays, declarando na mesma linha.

let carros=['Ferrari','Palio','Corola','Bmw'];
console.log(carros[2]);

Arrays, sendo declarado em bloco, mas o resultado é o mesmo.

let produto = [
'Computador',
'mouse',
'Teclado',
'Monitor'

] console.log(produto[2]);

Arrays de ingredientes, contendo outro array dentro de uma array.
Podemos ter infinitas arrays dentro de um array.

let ingredientes = [
['Arroz,', 'Macarrão','Feijão','Café'],
['Agua','Refrigerante','Suco'],
'Quantidade', 'Valor','Estoque'

] console.log(ingredientes[0]);
console.log(ingredientes[1]);
console.log(ingredientes[2]);
console.log(ingredientes);

Objetos

nome:'Fiat',
modelo: 'Uno',
peso:'800kg',
ligar:function(){
console.log("Ligando o carro: VRUM VRUM VRUUMMMMMMMM")
},

acelerar:function(){
console.log("Hrihhhh")
} };

console.log(carro2);
console.log(carro2.nome);
console.log(carro2.modelo);
console.log(carro2.peso);
carro2.ligar();
carro2.acelerar();
let carro2 = {
nome:'Fiat',
modelo: 'Uno',
peso:'800kg',
ligado:false,
ligar:function(){
this.ligado = true;
console.log("VRUMM VRUMM");
},

acelerar:function(){
if(this.ligado == true){
console.log("Hriiihhhhh");
} } }

console.log(carro2);
console.log(carro2.nome);
console.log(carro2.modelo);
console.log(carro2.peso);
carro2.ligar();
carro2.acelerar();
